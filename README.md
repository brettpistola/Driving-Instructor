# DrivingInstructorsAdelaide.com.au starter site (static)

This is a simple static HTML/CSS site you can deploy in minutes.

## What's inside
- Home page: `/`
- Instructors listing: `/instructors/`
- Instructor profiles: `/instructors/<slug>/`
- Suburb pages: `/suburbs/<suburb>/`
- For instructors: `/for-instructors/`
- Contact, Reviews policy, Privacy
- `sitemap.xml` and `robots.txt`

## Fast deployment options

### Option A: Netlify (recommended for forms)
1. Create a free Netlify account
2. Drag-and-drop this folder into Netlify "Deploys"
3. Set your domain in Netlify: **DrivingInstructorsAdelaide.com.au**
4. Point your domain DNS to Netlify (Netlify will tell you the exact records)

### Option B: Cloudflare Pages
1. Create a Cloudflare account
2. Upload via GitHub or direct upload
3. Add custom domain **DrivingInstructorsAdelaide.com.au**
4. Cloudflare will manage DNS

### Option C: GitHub Pages
1. Push files to a repo
2. Enable Pages (root)
3. Point DNS (A + CNAME) as instructed by GitHub
